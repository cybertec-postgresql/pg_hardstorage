# scenario to assert. Convert to scenarios if they grow real logic.

# Prints the embedded CHANGELOG; static output, nothing to assert.
changelog
# Spins up a throwaway PG 18 + Docker and runs a self-contained
# backup/restore demo; not a fit for the soak scenario harness.
demo
# Prints conceptual explanations; static informational output.
explain
# Prints the terminology glossary; static informational output.
glossary
# Validates pg_hardstorage.yaml and exits; config-shape check with
# no data path to exercise in a scenario.
lint
# Diagnostic that shells out to the configured LLM provider — the
# documented allow-list case (cannot run a real provider in CI).
llm doctor

# Read-only / diagnostic / maintenance leaves with no backup/restore
# data path of their own for a soak scenario to assert.

# Read-only display: prints a single backup's manifest JSON. The

# scenario; `show` only renders it, so there is no new data path.

# Diagnostic listing: walks $HSPLUGIN_PATH and probes each plugin,

# diagnostic leaves above.
plugin list
# Maintenance GC: sweeps approval requests past their TTL. The reaper

approval purge-expired
